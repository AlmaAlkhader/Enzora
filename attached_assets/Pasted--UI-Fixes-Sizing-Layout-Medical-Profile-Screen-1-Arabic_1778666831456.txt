### 2. Overall app sizing is wrong
The app appears to be rendering as if the 
screen is larger than it actually is.
Everything looks too big and overflows.

Fix:
- Set the root container max-width to 100%
- Make sure the viewport meta tag is correct:
  <meta name="viewport" content="width=device-width, 
  initial-scale=1.0, maximum-scale=1.0">
- All font sizes should be relative to screen 
  width, not fixed large values
- Reduce all font sizes by 15% across the app
- Make sure no container has a fixed width 
  larger than the screen width
- Use percentage widths or flex instead of 
  fixed pixel widths on containers
- Check that the preview is rendering at 
  mobile size (375px width) not desktop size
