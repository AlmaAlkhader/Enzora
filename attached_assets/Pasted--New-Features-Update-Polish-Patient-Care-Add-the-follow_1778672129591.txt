Card title: "Care Tips for You ✨"
Arabic: "نصائح العناية لك ✨"

"You are a wound care assistant for Enzora app.

YELLOW = safe, GREEN = early infection,

Current wound status: [yellow/green/blue]
Wound location: [location]
Days monitored: [number]
Language: [English/Arabic]"

- Soft shadow
- Light bulb icon 💡 at top

- Width: 220px, height: auto
- Horizontal scroll, snap to each card

Refresh tips button at top right of section.
Cache tips — only refresh when:
- Status changes
- User taps refresh
- New day starts

### 8. Mood Check-in

Once per day (first time user opens app that day):
Show a gentle bottom sheet modal:

Title: "How are you feeling today?"
Arabic: "كيف تشعر اليوم؟"

Four large emoji buttons in a row:
😊 Great  😐 Okay  😔 Not great  😣 In pain

On selection:
- Save to Firestore:
  /users/{uid}/moods/{date}
  { mood: "great"|"okay"|"notgreat"|"pain", 
    timestamp }
- Show warm response:
  😊 "That's wonderful! Keep it up 🌟"
  😐 "Take it easy today. You're doing great."
  😔 "We're here for you. Rest well today 💙"
  😣 "Please rest and contact your doctor 
       if the pain is related to your wound."
- Dismiss modal after response shown (2 seconds)

Show mood history on Profile screen:
- Last 7 days as colored dots
- 😊 = #06D6A0, 😐 = #FFB703, 
  😔 = #6B7FA3, 😣 = #EF233C
- Label: "Your wellness this week"
- Arabic: "صحتك هذا الأسبوع"

Do NOT show the modal if:
- Already checked in today
- App is in demo mode
- User has no active wounds

### 10. Offline Mode

Detect internet connection using NetInfo:
npm install @react-native-community/netinfo

When offline:
- Show amber banner at top of every screen:
  "You're offline – showing last saved data"
  Arabic: "أنت غير متصل – تعرض آخر بيانات محفوظة"
- Banner dismissible with X
- App still works with cached data
- Last Firebase readings cached in AsyncStorage
- Last wound data cached in AsyncStorage

When back online:
- Banner disappears automatically
- App syncs with Firebase immediately
- Show brief success message:
  "Back online – data updated ✓"
  Arabic: "عدت للاتصال – تم تحديث البيانات ✓"

Cache strategy:
- Cache last 50 readings per wound
- Cache full wound list
- Cache medical profile
- Cache last AI assessment
- Update cache every time new data arrives
