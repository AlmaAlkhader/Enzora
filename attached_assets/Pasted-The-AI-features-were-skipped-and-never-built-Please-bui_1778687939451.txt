### Setup

Use Replit's built-in Anthropic integration
via the API server proxy.
Do NOT embed the API key in the frontend code.

All Claude API calls must go through the 
Replit API server at:
POST /api/claude

Create server endpoint in the backend:
app.post('/api/claude', async (req, res) => {
  const { systemPrompt, userMessage, history } = req.body
  const response = await anthropic.messages.create({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 1024,
    system: systemPrompt,
    messages: [
      ...(history || []),
      { role: 'user', content: userMessage }
    ]
  })
  res.json({ text: response.content[0].text })
})

Frontend calls this endpoint:
async function callClaude(systemPrompt, userMessage, history = []) {
  try {
    const response = await fetch('/api/claude', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ systemPrompt, userMessage, history })
    })
    const data = await response.json()
    return data.text
  } catch (error) {
    console.error('AI error:', error)
    return null
  }
}
